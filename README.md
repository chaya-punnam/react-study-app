# React Study App

A simple and responsive Study App built using React.js.

## Features

* Clean and user-friendly interface
* React component-based architecture
* Responsive design
* Fast and lightweight application

## Technologies Used

* React.js
* JavaScript
* HTML5
* CSS3

## Installation

1. Clone the repository

```bash
git clone https://github.com/chaya-punnam/react-study-app.git
```

2. Navigate to the project folder

```bash
cd react-study-app
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm start
```
